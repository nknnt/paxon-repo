# Paxon Package Repository

Paxon is a lightweight package repository for tools such as **sopsh**, **development environments**, and other ecosystem components.

---

## Installation & Usage

Install a package using paxon:

```bash
paxon -S <pkgName>
```

## OS Build Variants

 + Windows Build: Stable and optimized production ready version.
 + macOS Build: Beta version including the latest experimental features.
 + Linux Build: Standard version balancing new features and stability.

# .spsh File Format

`.spsh` is the native scripting and configuration format used by sopsh.

It provides a minimal and flexible syntax for:
- Executing commands
- Configuring the shell environment
- Managing runtime behavior
- Defining shell preferences

An `.spsh` file can act as:
- A startup configuration file
- A reusable script
- A portable shell environment definition

---

# File Extension

```txt
.spsh
```
